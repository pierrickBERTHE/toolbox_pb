# toolbox_pb
Une boite à outil pour réaliser des tâches automatisables


## Configuration flags

- `LOG_TO_FILE`  
  Enable file logging.

- `ADD_CODEC_NAME_IN_OUTPUT`  
  Add video/audio codec names to output filename.

- `PRINT_ALL_KEYS_IN_METADATA_SUMMARY`  
  Print full FFprobe metadata instead of diff summary.
